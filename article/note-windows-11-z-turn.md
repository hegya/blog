# Windows11 折腾

- 激活 Windows11

Alt + X，选择“终端管理员”，依次输入三行命令：

```
slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX
slmgr /skms kms.03k.org
slmgr /ato
```

- 替换系统字体

  - 替换工具 https://github.com/Tatsu-syo/noMeiryoUI
