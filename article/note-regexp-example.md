# 正则范例

- 匹配汉字
  ```
  [\u3400-\u4DBF\u4E00-\u9FFF\uF900-\uFAFF] //所有汉字范围
  [\u2E80-\uFE4F] //所有CJK（中日韩）字符的范围
  ```
- EditPlus 中替换 “第138章此消彼长” 为 “第138章 此消彼长”
  ```
  查找项为：([0-9])([章])([\u3400-\u4DBF\u4E00-\u9FFF\uF900-\uFAFF])
  替换项为：\1\2 \3
  ```
- PHP 匹配多位数字
  ```
  preg_match_all('!\d+!', 被匹配, 匹配结果);
  ```
